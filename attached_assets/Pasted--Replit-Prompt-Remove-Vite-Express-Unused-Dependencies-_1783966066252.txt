# Replit Prompt — Remove Vite, Express & Unused Dependencies

The project has been migrated to Next.js. Vite, Express, and all related packages are no longer needed. This prompt cleans the project so only Next.js + Brevo remains.

Do NOT delete or touch anything inside `src/` — only remove build tools, server files, and unused packages.

---

## STEP 1 — Audit what exists first

Run these commands and report what you find before deleting anything:

```bash
# Show full folder structure (2 levels deep)
find . -maxdepth 2 -not -path "*/node_modules/*" -not -path "*/.git/*" -not -path "*/.next/*"

# Show all package.json files in workspace
find . -name "package.json" -not -path "*/node_modules/*"

# Check for Express references
grep -r "express" . --include="*.ts" --include="*.js" --include="*.json" -l --exclude-dir=node_modules

# Check for Vite references
grep -r "vite\|vitejs" . --include="*.ts" --include="*.js" --include="*.json" -l --exclude-dir=node_modules

# Check for database references (Drizzle, Postgres, Firebase Admin)
grep -r "drizzle\|postgres\|DATABASE_URL\|firebase-admin" . --include="*.ts" --include="*.js" --include="*.json" -l --exclude-dir=node_modules
```

Print a summary of everything found before proceeding.
