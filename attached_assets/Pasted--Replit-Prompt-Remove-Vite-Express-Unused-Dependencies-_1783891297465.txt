## What was removed and why

| Removed | Why |
|---|---|
| Express / api-server | Next.js API routes handle everything |
| Vite + vite.config | Next.js has its own bundler |
| pnpm-workspace.yaml | No longer a monorepo |
| Drizzle ORM + Postgres | No database needed for static site |
| Firebase Admin SDK | No database needed (Brevo handles emails) |
| lib/db, lib/api-zod, lib/api-client-react | Generated from old OpenAPI/Express setup |
| tsconfig.base.json | Next.js tsconfig is self-contained |
| pnpm-lock.yaml | Switching to npm for simplicity |

## What remains

| Kept | Purpose |
|---|---|
| Next.js 14 | Framework + SSG + routing |
| Brevo API route | Contact form email delivery |
| Tailwind CSS | Styling |
| TypeScript | Type safety |
| All src/ pages + components | The actual website |
