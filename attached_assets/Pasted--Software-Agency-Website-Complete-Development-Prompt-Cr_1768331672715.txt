# Software Agency Website - Complete Development Prompt

Create a modern, glassmorphism-themed software agency website with the following specifications:

## Design Requirements

### Visual Style
- **Primary Theme**: Pure glassmorphism design with frosted glass effects
- **Color Palette**: 
  - Soft gradients (blues, purples, teals, pinks)
  - Semi-transparent backgrounds with blur effects
  - White/light text on glass panels
  - Subtle neon accent colors for CTAs
- **Effects**:
  - Frosted glass cards with backdrop-filter blur
  - Soft shadows and glows
  - Smooth animations and transitions
  - Floating elements with subtle movements
  - Gradient borders on glass panels

### Layout Structure
- Responsive design (mobile, tablet, desktop)
- Smooth scrolling navigation
- Modern, spacious layout with breathing room
- Interactive hover effects throughout

## Technical Requirements

### Frontend
- HTML5, CSS3, JavaScript
- Use CSS variables for theme colors
- Tailwind CSS or custom CSS for styling
- Smooth scroll library (e.g., locomotive-scroll)
- AOS (Animate On Scroll) library
- Intersection Observer for animations

### Glassmorphism CSS Guidelines
```css
/* Example glass card styling */
.glass-card {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(10px);
  border-radius: 20px;
  border: 1px solid rgba(255, 255, 255, 0.2);
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.1);
}
```

### Backend Integration

**Brevo API Integration**:
- API endpoint for sending transactional emails
- Contact list management
- Email templates for confirmation and notifications

**Podio API Integration**:
- OAuth 2.0 authentication
- Create items in specific app
- Field mapping for all form data
- Error handling and retry logic

### Form Handling
- Client-side validation
- Server-side validation
- AJAX submission (no page reload)
- Rate limiting for spam prevention
- reCAPTCHA v3 integration
- Success/error notifications

### Performance
- Lazy loading images
- Optimized assets
- Minified CSS/JS
- Fast page load times
- Smooth animations (60fps)

### Diagrams for Service Pages
Use one of these approaches:
- **SVG-based diagrams**: Custom SVG with animated paths
- **Mermaid.js**: For flowcharts and architecture diagrams
- **Canvas-based**: Custom drawn diagrams with glassmorphism styling
- Include hover interactions to show additional information

## Notes for Replit
- Set up a Node.js backend for handling API integrations
- Use Express.js for server routes
- Store sensitive API keys in Replit Secrets
- Set up CORS properly for form submissions
- Test both Brevo and Podio integrations thoroughly
- Provide clear error messages for debugging
