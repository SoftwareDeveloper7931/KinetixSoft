# Replit Prompt ‚Äî KinetixSoft UI Polish Pass (Premium / Corporate)

Paste this into your Replit project. This is a **visual-only** pass ‚Äî don't change any copy, section structure, or page content from what's already built. Apply these tokens and component rules consistently across all 6 pages (Home, FlutterFlow, Podio, Lovable, Replit, Retool).

## Why this pass
The current look reads as a generic "dark AI SaaS" template ‚Äî near-black background, bright neon-blue accent, glow effects, rounded everything. The goal is the opposite impression: **precise, restrained, trustworthy** ‚Äî closer to how a serious engineering studio or boutique consultancy presents itself than a flashy AI startup.

## 2. Typography

- **Display face (headlines, H1/H2):** `Newsreader` (serif, Google Fonts) at weight 400‚Äì500. Use for the full headline now, not just the italic accent word ‚Äî but keep one phrase per headline in italic style as the existing accent pattern, just rendered in this serif rather than a script-like italic.
- **Body / UI face:** `Inter` for all body copy, nav, buttons, form labels, card titles.
- **Eyebrow / label face:** `Inter`, uppercase, letter-spacing `0.08em`, size `12px`, color `--text-secondary` ‚Äî used for the small numbered tags and section labels (e.g. "01", "FinTech", "Founder of Cashnix").

Type scale:
- H1: 48‚Äì56px / Newsreader 500 / line-height 1.1
- H2: 32‚Äì36px / Newsreader 500
- H3 (card titles): 20‚Äì22px / Inter 600
- Body: 16px / Inter 400 / line-height 1.6
- Caption/eyebrow: 12px / Inter 600, uppercase, letter-spaced

## 3. Shape & spacing

- Border radius: **6px** everywhere (buttons, cards, inputs, badges). Drop the larger pill-shaped radius currently on buttons and tags ‚Äî small, consistent radius reads more precise/corporate.
- Borders: 1px solid `--border` on every card and panel ‚Äî no shadow-only separation, no glow.
- Section vertical padding: increase to at least 96px top/bottom on desktop (more whitespace = more premium). 56px on mobile.
- Card internal padding: 32px (up from whatever's currently tighter).

## 4. Signature element: the ledger rule
Adopt one consistent divider motif used everywhere a section break or stat is shown ‚Äî a thin horizontal hairline with a small tick mark and index number, like an engineering spec sheet or financial ledger:

```html
<div class="ledger-rule">
  <span class="ledger-index">01</span>
  <span class="ledger-line"></span>
</div>
```
```css
.ledger-rule { display: flex; align-items: center; gap: 12px; margin: 16px 0; }
.ledger-index { font: 600 12px Inter; color: var(--text-secondary); letter-spacing: 0.08em; }
.ledger-line { flex: 1; height: 1px; background: var(--border); }
```
Use this above each major section heading (instead of just floating the heading on its own), and reuse the same tick-mark styling for the numbered service cards (01‚Äì06) so they visually connect to the same system. This is the one recognizable signature ‚Äî keep everything else quiet around it.
