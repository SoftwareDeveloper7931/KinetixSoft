# Software Agency Website

## Overview
Modern glassmorphism-themed software agency website with Brevo and Podio integrations.

## Tech Stack
- Frontend: React, Tailwind CSS, Framer Motion (Glassmorphism)
- Backend: Node.js, Express, PostgreSQL
- Integrations: Brevo (Email), Podio (CRM)

## Features
- Glassmorphism UI
- Service pages (Podio, FlutterFlow, Retool)
- Contact form with external integrations

## Environment Variables
- `BREVO_API_KEY`: For sending emails
- `PODIO_...`: For CRM integration
